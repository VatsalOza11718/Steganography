About BIS

A practical overview of the platform and its core methods.

What BIS Does

BIS hides secret text inside ordinary text, audio, video, or images. You can optionally protect the hidden payload with a password so extracted data is decrypted only by authorized recipients.

Steganography Methods

Text in Text

Encodes hidden bits with zero-width Unicode characters while preserving visible cover text.

Text in Audio

Uses LSB embedding in WAV samples to store message data in a way that is generally inaudible.

Text in Video

Distributes hidden data across video frames using LSB-style embedding for larger payload capacity.

Text in Image

Applies image-based embedding and supports ML-assisted mode when dependencies are available.

Security Notes

  • Optional password protection uses authenticated encryption (AES-GCM).
  • Steganography hides existence of data; encryption protects content confidentiality.
  • If media is aggressively re-encoded or edited, extraction reliability may decrease.
